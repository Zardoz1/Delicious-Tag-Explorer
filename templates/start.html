Welcome to my Delicious Tag Visualizer. This application is designed to show the relationship between your tags. For any tag, it show the other tags that appear with it, and the number of times the two tags appear together. This application only uses the public interface to Delcious. There is no need to enter your password or provide any information other than your user name.

Type in your Delicious user name here and hit OK. Your tag list will be fetched from Delicious. Then you can select one of your tags to start the visualization process. You will be able to "drill down" thtough the tag relationships by extending the graph from that starting tag.

Please be patient when using this application. Fetching tags and other data from Delicious is not exactly internet-fast, for some reason. That is why the application works by building the graph from a start tag. Also the graphing package that I am using slows down as the graph gets bigger. I recommend using this app to explore reasonably closely related tags so that the graph does not get too big
